# MODULE 2 — Team & Founders Deep OSINT

## Step-by-step verification protocol

### 1. Identity Verification
```
For each named founder/team member:
□ Full name → Google search + LinkedIn
□ LinkedIn created: before or after project launch?
□ Employment history: verifiable previous employers?
□ Education: verifiable institutions?
□ Photo: reverse image search (Google Images, TinEye, Yandex)
□ Is the photo a stock image? (check Shutterstock, Getty)
□ Other social accounts: consistent persona across platforms?
```

### 2. Track Record Investigation
```
□ Previous company: registered? Still active? 
□ Previous projects in crypto/MLM: what happened to them?
□ Trustpilot/BBB: complaints about previous ventures?
□ Court records: litigation, bankruptcy, fraud charges?
   - US: PACER, Bloomberg Law
   - UK: Companies House, HMRC
   - RU: kad.arbitr.ru, ras.arbitr.ru, fssp.gov.ru
   - EU: local commercial court registries
□ Regulatory sanctions: SEC/FCA/AMF enforcement actions?
```

### 3. Alias Detection
```
□ Search founder name + "scam" / "fraud" / "lawsuit"
□ Search founder name + previous company names
□ Check if same photo appears with different name
□ Bitcointalk.org posts (crypto): username history
□ BehindMLM.com: prior MLM involvement
```

### 4. Team Size Reality Check
```
Claimed team size on website: N
LinkedIn employees listed: M
GitHub contributors (if open source): K

If N >> M by 3x or more → inflated claims
If K = 0 for "tech project" → code not being written
```

### 5. Advisor Legitimacy
```
For each listed advisor:
□ Did they publicly confirm this role? (tweet, LinkedIn post)
□ Is their name used without permission? (contact them)
□ Are they active in crypto/MLM space or random celebrities?
□ Advisor with 0 posts about the project they "advise" = red flag
```

## Scoring
- Fully verified, clean-record team: 13–15 pts
- Partially verified, no red flags: 8–12 pts
- Anonymous but with track record: 5–8 pts
- Anonymous, no track record: 3–5 pts
- Verified but prior scam involvement: 0–2 pts
- Confirmed fake identities: 0 pts + escalate to Critical Finding
